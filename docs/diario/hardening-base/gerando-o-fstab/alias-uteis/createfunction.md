# createFunction

```
alias createFunction='
FUNCTION="/tmp/fnt/fn.sh"
rm -f $FUNCTION
nano $FUNCTION
chmod +x $FUNCTION
sudo $FUNCTION
'
```

O objetivo desse alias é criar um arquivo dentro de um tmpf para executar blocos inteiros, onde ao invés de focar em ficar dando varias(lá ele) vezes o mesmo comando, foco em criar um script novo, e executa-lo, removendo o aterior.
